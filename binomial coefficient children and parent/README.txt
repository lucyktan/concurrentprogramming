Answers to hw questions:
1. 
cuserid() is the username of the Linux account.
Real user id is the id that created the process.
Effective user id is the id that evaluates privileges of the current user id that is executing the processes.
Group id is the id of a group of processes that can be controlled together.

2. 
Processes may observe zero time values because the process execution is quicker than can be detected and the default CPU time is 0.
